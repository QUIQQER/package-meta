QUIQQER SEO - META - Permalink Plugin
========

SEO - META - Permalink Plugin for QUIQQER

Package Name:

    quiqqer/meta


Features
--------

- META data can be set for pages individually
- Permalinks can be set for pages individually
- SEO help

Installation
------------
